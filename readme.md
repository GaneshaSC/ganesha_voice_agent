# Ganesha Voice Agent (Minimal Version)

A lightweight FastAPI backend for handling Twilio Voice WebSocket connections.  
This version intentionally excludes STT, TTS, Deepgram, ElevenLabs, and Azure.  
It provides a clean foundation for future audio processing or AI voice features.

---

## 🚀 Features

- FastAPI backend
- Twilio Media Streams WebSocket endpoint
- OpenAI LLM integration (text-only placeholder)
- No audio processing (STT/TTS removed)
- Simple, stable, production-ready structure
- Railway deployment compatible
